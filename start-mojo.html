0.7 27-Jun-2014

jcabi-mysql:start

Full name:

com.jcabi:jcabi-mysql-maven-plugin:0.7:start

Description:

Starts MySQL.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 0.1.
  • Binds by default to the lifecycle phase: pre-integration-test.

Required Parameters

Name Type Since Description
data File 0.1 Location of MySQL data.
Default value is: ${project.build.directory}/mysql-data.
dist File 0.1 Location of MySQL distribution.
Default value is: ${project.build.directory}/mysql-dist.

Optional Parameters

Name Type Since Description
dbname String 0.1 Database name to use.
Default value is: root.
options List 0.1 Configuration options.
password String 0.1 Password to use.
Default value is: root.
port int 0.1 Port to use.
Default value is: 3306.
skip boolean 0.1 Shall we skip execution?
Default value is: false.
user String 0.1 Username to use.
Default value is: root.
