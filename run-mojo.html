jcabi-mysql:run

Full name:

com.jcabi:jcabi-mysql-maven-plugin:0.8.1:run

Description:

Run MySQL in background and don't stop it when Maven is finished.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 0.4.
  • Binds by default to the lifecycle phase: pre-integration-test.

Required Parameters

Name Type Since Description
data File 0.4 Location of MySQL data.
Default value is: ${project.build.directory}/mysql-data.
dist File 0.4 Location of MySQL distribution.
Default value is: ${project.build.directory}/mysql-dist.

Optional Parameters

Name Type Since Description
clearexistingdata boolean 0.4 Shall we always delete an existing database or reuse it?
Default value is: true.
dbname String 0.4 Database name to use.
Default value is: root.
options List 0.4 Configuration options.
password String 0.4 Password to use.
Default value is: root.
port int 0.4 Port to use.
Default value is: 3306.
skip boolean 0.4 Shall we skip execution?
Default value is: false.
user String 0.4 Username to use.
Default value is: root.
